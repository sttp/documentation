Streaming Telemetry Transport Protocol Documentation

Home View on GitHub

The Streaming Telemetry Transport Protocol (STTP) is optimized for the demands of transporting high volume continuous streaming data and related metadata. The protocol allows for the transmission of any information that can be represented longitudinally, e.g., time-series data. STTP is currently undergoing IEEE standardization (P2664), see: https://standards.ieee.org/ieee/2664/7397/
